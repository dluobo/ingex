XDCAM files

XDCAM MXF files are OP-1a, where each track of video and audio are interleaved and stored in a single MXF file.

When a new clip is recorded the OP-1a MXF is stored with a filename containing an incrementing clip number. In addition, an XML file is written and a browse-quality MXF file. An example of the names given to these files is shown below: 

C0012.MXF
C0012S01.MXF
C0012M01.XML
